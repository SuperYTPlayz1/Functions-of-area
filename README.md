# 🌀 Functions Of Area (FOA)
### *Next-Gen Region Automation for Minecraft Servers*

[![Version](https://img.shields.io/badge/Version-0.1.0-blue.svg)](https://modrinth.com/plugin/functions-of-area)
[![License](https://img.shields.io/badge/License-GPL%20v3-green.svg)](https://www.gnu.org/licenses/gpl-3.0)
[![Platform](https://img.shields.io/badge/Platform-Paper%20/%20Spigot-orange.svg)](https://papermc.io)

**Functions Of Area (FOA)** is a high-performance, fully GUI-based region management tool. Built for modern servers, it allows you to define 3D zones and trigger custom actions (Commands/Messages) with beautiful on-screen countdowns.

---

## ✨ Features
*   **🖥️ 100% GUI Based:** No complex commands. Create, Edit, and Manage everything via menus.
*   **🪓 Advanced Wand:** Precision selection with the Golden Wand.
*   **⏳ Dynamic Timers:** Supports Seconds, Minutes, Days, and even Months.
*   **🎨 Custom Visuals:** 5 vibrant colors for on-screen Title countdowns.
*   **⚡ Smart Execution:** Auto-triggers player commands and messages.
*   **📖 Paginated List:** Manage hundreds of areas with ease.
*   **📍 Instant Teleport:** One-click TP to any area's center.
*   **🛡️ Cancellation Logic:** Auto-stops if a player leaves the area mid-countdown.

---

## 📖 How To Use
1.  **Get the Wand:** Type `/foa` and take the **FunctionsArea Wand**.
2.  **Define Area:** 
    *   **Right-Click** for Position 1.
    *   **Left-Click** for Position 2.
3.  **Open Menu:** **Crouch (Shift) + Click** with the wand.
4.  **Configure:** Use the **Golden Pickaxe** to set Name, Timer, and Actions.
5.  **Build:** Click the **Build** button to activate the zone.
6.  **Manage:** Access the **Book icon** to Edit, Teleport, or Delete zones.
